Sobre mi

¡Hola! Soy Ezequiel Oliver, tengo 32 años y soy oriundo de la ciudad de La Plata, Buenos Aires, Argentina. Soy un apasionado del desarrollo web y del análisis de datos. Aunque cada uno de estos campos tiene sus propias complejidades, en ambos, de una manera metafórica, se trata de crear sentido a partir del caos: ya sea mediante el diseño de una interfaz limpia y funcional, o la destilación de conocimientos a partir de conjuntos de datos en bruto.
Como desarrollador web, estoy orientado principalmente al front end, con habilidades para crear interfaces de usuario atractivas y funcionales. Mi experiencia abarca HTML, CSS , JavaScript y Bootstrap, junto con un toque de backend usando Python y Flask para asegurar que cada proyecto sea completo y bien integrado. Esto me permite entender y colaborar en ambos lados del desarrollo web.
Por otro lado, como analista de datos, tengo experiencia en el análisis exploratorio y la visualización de datos, utilizando herramientas como Python (y sus diversas librerías), SQL y Looker Studio. También he adquirido conocimientos en machine learning, lo que me permite abordar problemas complejos y extraer valor de los datos.
Además de mis competencias técnicas, soy una persona curiosa y perseverante, orientada a la resolución de problemas. Me apasiona seguir aprendiendo y enfrentando nuevos desafíos, ya sea desarrollando soluciones web o analizando datos para encontrar respuestas. Estoy listo para llevar mis habilidades al siguiente nivel en proyectos que combinen creatividad y lógica.
